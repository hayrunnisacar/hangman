# Pendu
Un jeu de pendu réalisé avec **React**, **TypeScript** et **Vite**.

## Features
- Jeu du pendu classique
- Système d'indice

## Installation et lancement
- Clone le dépôt et installe les dépendances :
  ```
  git clone [https://github.com/hayrunnisacar/hangman.git](https://github.com/hayrunnisacar/hangman.git)
  cd hangman
  npm install
  npm run dev
  ```

* Back-end (API)
```
git clone [https://github.com/alexischarp-education/node-hangman-api.git](https://github.com/alexischarp-education/node-hangman-api.git)
cd node-hangman-api
npm install
npm run dev
```
